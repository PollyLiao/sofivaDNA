DNA親子鑑定

每個人都有23對（46條）染色體染色體是由DNA去氧核醣核酸所組成
精子卵子結合成受精卵，生長出一個新的生命個體，會繼承父母一半的染色體

Description

每個人都有23對（46條）染色體，染色體是由DNA（去氧核醣核酸）所組成。而男性的精子和女性的卵子各有23條染色體，當精子和卵子結合形成受精卵，生長出一個新的生個體，這個新生命就會從父親繼承一半的染色體，另一半則是從母親獲得。

親子鑑定檢測是檢驗子女的DNA，是否帶有來自父母方的基因片段。基因片段如果完全一致，可確立親子關係；如果有3個以上的點位不同，可排除親子關係；若是1-2個點位不同，可能為基因突變，須增加基因檢測點位以利辨別。本檢測否定親子關係的準確率近乎100%，肯定親子關係的準確率可達99.99%。

人的基因點位是有可能突變的，根據統計，突變的機率約1/1,000，所以如果比對的結果是只有1-2個點位不合，應該考慮是否為基因突變的問題，這時候就需要再檢測更多的基因點位，或是透過其他的輔助鑑定系統 註，來排除或確認親子關係。

註輔助的鑑定系統

如：檢測性染色體（Y染色體），或細胞質中的粒腺體DNA。

紅血球血型測試

A、B、AB及O型等4種類型
雖然不能作為親子關係的依據，但具有一定的參考價值

example

有一對父母，父親的血型是A型，母親的血型是B型他們所生的子女，可能的血型為A、B、AB及O型都有可能完全無法鑑別親子關係

以單一血型系統來判斷親子關係出錯的機會很高，若檢驗的血型系統越多，則作為親子鑑定的準確性就會越高，但是結果也僅能夠作出否定親子關係的結論，不能完全確定是親子關係

Description

父母與子女血型的可能性

以單一血型系統來判斷親子關係出錯的機會很高，若檢驗的血型系統越多，則作為親子鑑定的準確性就會越高，但是結果也僅能夠作出否定親子關係的結論，不能完全確定是親子關係。

人類白血球組織抗原

白血球的血型系統，稱為人類白血球組織抗原，簡稱 HLA
Human Leukocyte Antigen

白血球也是有血型的，所以也可以用來鑑定親子關係

由於HLA為人類基因體最多形態的系統，所以HLA開始被用在親子關係的鑑定
鑑別度較以往大幅提高，再結合血型檢驗，則能將準確度再提高

Description

雖然 HLA 的變化性很高，但是沒有血緣關係的人仍就有可能出現相同的型別，因此部分個案還是無法鑑別親子關係。

染色體多態性鑑定

染色體的多態性是可遺傳的，1970年代開創了使用染色體多態性鑑定親子關係的技術
染色體的多態性鑑定親子關係，倚靠的是技術人員的主觀判斷

Description

正常人的染色體常有各種形態的微小變異，像是隨體註增大、重複或缺少、染色體大小形態或著色等方面的變異，稱為染色體多態性又稱異態性（heteromorphism）。

染色體的多態性是可遺傳的，所以在1970年代開創了使用染色體多態性鑑定親子關係的技術。因為是利用形態來鑑定親子關係，倚靠的是技術人員的主觀判斷，且易有人為誤差，準確率不盡如人意，當然也就無法普及而被自然淘汰了。

註：隨體（satellite），位於染色體末端的、圓形或圓柱形的染色體片段。
